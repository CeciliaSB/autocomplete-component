
# Auto-complete App

This an auto-complete app created with a react-typescript template.


## Run Locally

Clone the project

```bash
  git clone https://github.com/CeciliaSB/autocomplete-component
```

Go to the project directory

```bash
  cd autocomplete-component
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
